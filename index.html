Voidless
Changelog

A Markdown editor that stays out of your way.

Download for macOS Download for Windows

Tauri 2 + React 19 + CodeMirror 6

Features

Live preview

  • Inline rendering, no split pane — markdown renders right in the text.
  • Reveal on caret: move the cursor into markup to see the raw syntax; leave and it renders again.
  • Works line/block-wise, doesn't repaint the whole document.

Basic markup

  • Headings H1–H3 (styled sizes).
  • Bold **text**, italic *text*.
  • Inline code `code`.
  • Horizontal rule --- → rendered as <hr>.
  • Blockquotes.
  • Links — rendered as a compact widget (see Links & URLs).

Extended markup

  • Highlight ==text== → yellow marker (== hidden).
  • Strikethrough ~~text~~ (Cmd/Ctrl+Shift+X).
  • Wiki-links [[filename]] — brackets hidden, clickable, autocomplete over vault files (incl. table cells).
  • Alarms @⏰ HH:MM — always highlighted red.
  • Images with width ![alt|400](path) — width in px; relative paths and Windows drive-letter paths.

Lists & tasks

  • Bulleted lists with smart Enter (continues the item).
  • Numbered lists: hierarchical numbering (1, 1.1, 1.2…), labels recomputed automatically.
  • Checkboxes / task lists - [ ] / - [x]: clickable widget; done items struck through; smart Enter continuation.
  • List indent: Tab / Shift+Tab, correct for multi-line selection.
  • Move lines: Ctrl+↑ / Ctrl+↓.

Tables (inline GFM)

  • Rendered as an interactive table right in the editor.
  • Edit cells by click; add/remove columns and rows.
  • Drag rows by grip; resize columns.
  • Cell-range selection, delete row range.
  • Wiki-links inside cells — clickable.
  • Column config (option chips): RMB on header → popover for bulk-fill and clear.
  • Column alignment (left/center/right).

Code & embedded blocks

  • Fenced code blocks with info-string (language); nested fences via a longer outer fence.
  • Copy button on code blocks.
  • Built-in render blocks: excalidraw (embedded Excalidraw diagrams) and univer (embedded spreadsheet, UniverJS).
  • Block-renderer system extensible by plugins (a plugin registers its own language).

Links & URLs

  • Links rendered as a compact label instead of the long URL.
  • Smart URL shortening: domain: last-segment; special cases — Jira → Jira: KEY-123, Habr → Habr: slug.
  • Expands when the caret enters.

Frontmatter (YAML)

  • --- … --- block at the top — concealed/folded in render.
  • Frontmatter panel: edit key-value fields in a form.
  • List fields [a, b, c] → recognized as a select with options.
  • Special key _fileName in a template's frontmatter sets the file name and is stripped from content.

Alarms (reminders)

  • Syntax: @⏰ 18:30 (today), @⏰ 10.06 09:00 (this year), @⏰ 10.06.2027 09:00 (specific date).
  • Checker every 30s → red in-app banner + chime.
  • Reveal: click the notification to jump to the alarm line.
  • Snooze (in-memory), fired-registry (~1h window against repeats).
  • Insert via / menu or Command palette.

Editor: behavior & input

  • Autosave (debounced); flush before switching files and on close.
  • Undo/Redo, find in document (Cmd/Ctrl+F).
  • Line numbers (toggle); editor font size and UI zoom (separate).
  • Smart paste: collision-aware path resolution for pasted images/files.
  • Command menu / at line start: quick inserts.
  • Autocomplete for wiki-links [[.

Command palette (Cmd/Ctrl+P)

  • Insert Checkbox / List / Table / Code Block.
  • Insert Date / Insert Time.
  • Toggle Strikethrough; Insert Alarm (⏰).
  • Open Flashcards (Anki); Open Web Parser.
  • Insert Excalidraw / Insert Univer.
  • + plugin commands (dynamic).

Keyboard shortcuts

  • Global: Cmd+P palette · Cmd+F search · F1 templates · Cmd+W close tab · Cmd+Shift+T reopen tab · Cmd+Alt+←/→ history · Cmd+R/F5 reload · Cmd+Alt+I devtools.
  • Editor: undo/redo, find, strikethrough, indent, smart Enter, / menu.
  • Sidebar: Cmd+X cut · Shift+click range · Alt+click right split · Cmd+click new tab · RMB context menu.
  • Mouse side buttons — back/forward in history.

File manager / sidebar

  • Vault = root folder; recursive tree scan.
  • Create file, folder, Base file (.base).
  • Rename, delete (incl. multiple), move (drag).
  • Cut / paste files: Cmd+X → RMB "Paste here".
  • Multi-select (Shift+click range).
  • Colors & icons for files/folders (persisted + synced).
  • Sort by name / created date; collapse/expand all; sidebar search.
  • Folder templates: a template auto-applies when creating a file in a mapped folder.

Tabs, splits, navigation

  • Tabs per pane; close, reorder by drag; reopen closed tab (Cmd+Shift+T).
  • Splits (panes): flexible layout tree, vertical/horizontal.
  • Drag a tab to an edge → new split; to center → move as tab.
  • Resize dividers; open in new tab / right split.
  • Per-pane navigation history (back/forward, mouse side buttons).
  • Session restored on startup (panes, tabs, active files, layout).

Base — notes as a database

  • Aggregates note frontmatter from a folder into a table/board (.base files).
  • Table view: columns from frontmatter, sort by column.
  • Cards / Kanban view: group by a field, custom group order.
  • Folder filter (include/exclude), record filter.
  • Bulk-fill: mass-fill a column's value via popover.
  • Click a record to open its source file.

Google Drive sync

  • OAuth2 PKCE, local redirect on port 1827.
  • Scope drive.file (non-sensitive — no Google verification, no test-user cap).
  • Auto-sync on start and every 5 minutes; works offline.
  • Move detection — no duplicates across devices (flip-flop protection).
  • Remote-edit detection via MD5 without downloading.
  • Conflict resolution: keep original / keep conflicting copy.
  • Vault-meta (colors/icons/settings) synced across devices.

Plugins

  • Plugin system: plugins from the .plugins folder inside the vault.
  • A plugin is arbitrary JS with app/Tauri access → requires user consent (once per plugin).
  • Plugins can add commands, block renderers (own languages), react to vault changes.
  • Reload plugins in settings.

Anki — flashcard trainer

  • Built-in, full-view. Store: <vault>/.anki/cards.json.
  • SRS (spaced repetition): interval/reps/ease, lapse logic, due cards.
  • Decks: create/delete, filter by deck.
  • Modes: Review / Add / Cards / Stats.
  • Auto-translate via DeepL: RMB on a word → "Add to Anki".
  • Front/Back languages configurable; DeepL key stored locally.

Web Parser — link extractor

  • Built-in, full-view. Generic link extractor: sources (any sites) + keyword filters → pulls matching links (jobs, news, releases…).
  • Sources: <vault>/Web Parser/sources.md.
  • Run stats: parsed / JS-skipped / skipped / 404 / 403.
  • Marks new results (vs previous run).
  • CSV export, run log.

Templates

  • Templates folder (.md files), opened with F1.
  • Variables: {{date}}, {{dateRU}}, {{time}}, {{week}} (Mon–Fri of the week).
  • _fileName in frontmatter sets the file name.
  • Folder templates — auto-applied per folder.
  • Template field editor; sync vault to template.

Settings

  • Tabs: Editor / Plugins / Shortcuts / About.
  • Editor: font size, UI zoom, line numbers, templates folder, folder templates, help references.
  • Plugins: loaded plugins, built-in blocks, reload, Anki/DeepL settings.
  • Shortcuts: full reference (macOS + Windows).
  • About: version, license, third-party libraries.

System

  • Auto-update: checks on startup → "Update available" banner with install + restart.
  • Date-based versioning YY.M.D.BUILD.
  • File watcher: tracks on-disk changes; guards against echo-saves (iCloud/sync folders).
  • Diagnostic log in <vault>/.artifacts.
  • Persistence via Tauri store; CSP enabled.
