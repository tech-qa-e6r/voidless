# Voidless — лендинг

Статический лендинг для распространения Voidless. Чистый HTML/CSS, без сборки.
Публикуется на GitHub Pages → https://voidless.app/

## Запуск локально
Просто открой `index.html` в браузере (двойной клик) — всё работает.

## Структура
- `index.html` — страница
- `style.css` — стили в теме редактора
- `assets/screenshot.png` — скриншот редактора (положить вручную)

## Что доделать
- Положить реальный скриншот в `assets/screenshot.png`.
- Когда выйдет первый релиз — повесить кнопки «Скачать» на
  `github.com/tech-qa-e6r/voidless/releases/latest`.

## Деплой
GitHub Pages из корня ветки `main`, домен `voidless.app` (CNAME).
Push в `main` → сайт обновляется автоматически.
